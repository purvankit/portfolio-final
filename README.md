# Developer Portfolio

## Overview
This is a futuristic, terminal-themed developer portfolio built with HTML, CSS (Bootstrap Grid), and Vanilla JavaScript. It features a "Cyberpunk Terminal Hybrid" aesthetic with neon colors, glitch effects, and interactive elements.

## Features
- **Boot Sequence**: Typewriter effect on load simulating a system startup.
- **Theme Toggle**: Switch between "Dark Mode" (Default) and "Light Mode" (Day/Cream theme). Persists via LocalStorage.
- **Interactive Terminal**:
    - Draggable-style windows (visual only).
    - Command-line style contact form with validation.
    - Live clock and system status.
- **Responsive Design**: Fully responsive layout using Bootstrap Grid and custom media queries. mobile-friendly navigation.

## Setup
1. Unzip the folder (if zipped).
2. Open `index.html` in any modern web browser.
3. No server required!

## Technologies
- HTML5
- CSS3 (Variables, Keyframes, Flexbox/Grid)
- JavaScript (ES6+)
- Bootstrap 5 (Grid Only)
- Google Fonts (IBM Plex Mono)
