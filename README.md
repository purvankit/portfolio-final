# Purvankit Khadatkar | Cyberpunk Developer Portfolio

## Overview
A futuristic, terminal-themed developer portfolio built with semantic **HTML5**, modern **CSS3**, **Bootstrap 5 (Grid)**, and **Vanilla JavaScript**. The design features a "Cyberpunk Terminal Hybrid" aesthetic with neon colors, glitch effects, interactive terminal windows, and a dark/light mode toggle.

## Live Demo features
- **Terminal Aesthetics**: Pure black background, neon cyan/magenta text, scanlines (CSS), and CRT-style overlays.
- **Boot Sequence**: Typewriter effect on page load simulating a system startup.
- **Theme Toggle**: 
  - **Dark Mode** (Default): High contrast neon on black.
  - **Light Mode**: Cream/Beige terminal paper style with solid purple accents for readability.
- **Interactive UI**:
  - Window-style containers for sections.
  - Hover effects on project cards and buttons.
  - Custom cursor and glitch text animations.
- **Responsive Design**:
  - Desktop: Sticky navbar with Resume download.
  - Mobile: Hamburger menu, full-width touch targets, optimized layout (Resume download moves to Contact section).
- **Contact Form**: Terminal-style input fields with simulated transmission and client-side validation.

## Setup & Usage
1. **Clone/Download** the repository.
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari).
3. No build step or backend server required—it runs entirely client-side.

## Technologies
- **Frontend**: HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
- **Layout**: Bootstrap 5 Grid System + CSS Flexbox.
- **Fonts**: 'IBM Plex Mono' (Google Fonts).
- **Icons**: SVG Icons (Custom).

## Customization
- **Personal Details**: Edit the `#about` section in `index.html`.
- **Projects**: Update the `.project-window` blocks in `#projects`.
- **Resume**: Replace `Resume/Resume_purvankit.pdf` with your own file.
